B 软件使用

如第 1 章所述，这本书并不是一本全面的 knitr 或 rmarkdown 指南。在本章中，我们简要地解释了 knitr 和 rmarkdown 中的一些基本概念和语法。如果你有更多的问题，可以在 StackOverflow (https://stackoverflow.com) 上提问，并用 r、knitr、rmarkdown 和/或 bookdown 等任何适合的标签标记你的问题。
